# 기능별 암기장
{: .fs-7 }

로그인한 사용자 본인 홈에 등록된 암기장목록을 기능별로 제공합니다.

## 반복학습

반복학습으로 지정된 암기장 목록을 조회합니다.

최대 20개 까지만 지정 조회 가능합니다.

## 즐겨찾기

즐겨찾기로 지정된 암기장 목록을 조회합니다.

## 구독암기장

본인이 구독중인 암기장 목록을 조회합니다.

## 공개암기장

본인이 공개한 암기장 목록을 조회합니다.

## 전체보기

본인 홈의 전체 암기장 목록을 조회합니다.
