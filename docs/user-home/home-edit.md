# 사용자 홈 관리
{: .fs-7 }

회원 본인의 홈 관련 설정을 하게 됩니다.

홈이름, 소개글, 태그, 아바타, 공개여부 등을 설정하실 수 있습니다.

홈을 공개로 설정하실 경우, 다른 사용자들에게 노출됩니다.

작성하신 태그는 메뉴로도 사용되며, 암기장 작성시 등록하신 태그를 선택하게 됩니다.

홈 태그는 최대 10개 까지 설정 가능합니다.

방문하신 이용자들이 태그 분류별로 암기장을 조회할수 있게 됩니다.
