# Sign in
{: .fs-7 }

SNS 계정을 이용한 로그인 기능을 제공합니다.

{: .note }
개인정보 보호를 위해 공용 PC에서는 사용 후 SNS 계정의 로그아웃 상태를 반드시 확인해주세요.
공용 PC 에서는 브라우저의 시크릿모드(InPrivate) 사용을 권장합니다.

![sign-in](/assets/images/sign-in.png)

## 회원가입

최초 로그인 시 사용하실 홈아이디를 입력하고, 약관에 동의하시면, 회원가입이 완료됩니다.
