# 간편 암기장
{: .fs-7 }

간편하게 카드 앞,뒷면에 해당하는 질문/답변을 여러줄 작성하여 Flashcards 를 보여줍니다.

카드를 클릭하면 뒷면의 답변을 확인하실 수 있습니다.

뒷면 먼저보기, 섞어보기 기능을 제공합니다.

회원가입 없이 이용하실 수 있는 기능입니다.
